# 智能体提示词模板使用说明

## 📋 功能概述

现在你可以在创建或编辑智能体时，使用**提示词模板库**快速配置专业的系统提示词，无需手动编写复杂的提示词。

## 🎯 如何使用

### 步骤1：打开智能体编辑器
1. 进入后台管理页面
2. 点击「智能体」标签
3. 点击「创建自定义智能体」或编辑现有智能体

### 步骤2：选择模板
1. 在「Logic / 逻辑」区域，点击 **「模板库」** 按钮（蓝色按钮，带书本图标）
2. 会弹出模板选择窗口
3. 选择你需要的模板（例如：**文案选题助手**）
4. 点击模板卡片即可应用

### 步骤3：自动填充
选择模板后，系统会自动填充：
- ✅ **系统提示词**：完整的专业提示词
- ✅ **智能体名称**：模板对应的名称
- ✅ **描述**：模板的描述信息
- ✅ **分类**：模板的分类（如 Marketing）

### 步骤4：自定义调整（可选）
应用模板后，你可以：
- 修改系统提示词，添加你的特定需求
- 调整智能体名称和描述
- 添加样式预设（Styles）
- 上传知识库文件

### 步骤5：保存
点击「保存」按钮，智能体创建完成！

## 📚 当前可用模板

### 1. 文案选题助手
- **适用场景**：短视频文案选题、内容策划、爆款选题生成
- **核心能力**：
  - 8大选题元素（成本、人群、猎奇、最差、反差、怀旧、荷尔蒙、头牌）
  - 多元素组合策略（2-4个元素组合）
  - 质量保证清单
  - 元素兼容性矩阵

**使用示例**：

**传统模式（不推荐）**：
```
用户：帮我生成关于"美妆护肤"的短视频选题
智能体：直接生成选题（可能不够精准）
```

**交互式模式（推荐）**：
```
用户：我要写一篇文案

智能体：
【信息收集进度】
✅ 已收集：无
❌ 还缺少：目标受众、行业/产品、核心目标、情绪弧线、内容类型

你好！我很乐意帮你生成文案选题。为了给你最精准的方案，我需要了解一些信息。

首先，请问你的目标受众是什么？
例如：30-40岁职场女性、大学生、宝妈、创业者等

用户：30到40岁的职场女性

智能体：
【信息收集进度】
✅ 已收集：目标受众（30-40岁职场女性）
❌ 还缺少：行业/产品、核心目标、情绪弧线、内容类型

好的，目标受众是30-40岁职场女性。

接下来，请问这个文案是关于什么行业或产品的？
例如：美妆护肤、教育培训、科技产品、生活方式等

用户：美妆护肤

智能体：继续提问下一个问题...

（直到收集完所有信息后）

智能体：
【信息收集完成 ✅】

根据你提供的信息：
- 目标受众：30-40岁职场女性
- 行业：美妆护肤
- 核心目标：提升品牌认知
- 情绪弧线：好奇→共鸣→向往
- 内容类型：短视频脚本

我为你生成以下选题方案：
【选题1】...
【选题2】...
```

**优势**：
- ✅ 一问一答，逐步收集信息
- ✅ 显示收集进度，用户清楚知道还缺什么
- ✅ 生成的内容更精准，完全匹配用户需求
- ✅ 类似"记忆库"，AI会记住已收集的信息

## 🔧 模板工作原理

### 系统提示词的作用
当你选择模板后，模板中的系统提示词会被设置为智能体的 `system_prompt`。当用户与智能体对话时：

1. **用户消息** → 发送给 AI
2. **系统提示词** → 作为 AI 的"角色设定"和"工作指南"
3. **AI 响应** → 根据系统提示词中的规则和策略生成内容

### 为什么需要模板？
- ✅ **专业性**：模板包含经过验证的专业策略和规则
- ✅ **一致性**：确保智能体始终遵循最佳实践
- ✅ **效率**：无需从零开始编写复杂的提示词
- ✅ **可扩展**：可以轻松添加更多模板

## 🎨 自定义模板

如果你想添加自己的模板，可以编辑 `utils/promptTemplates.ts` 文件：

```typescript
export const PROMPT_TEMPLATES: PromptTemplate[] = [
  {
    id: 'your-template-id',
    name: 'Your Template Name',
    nameZh: '你的模板名称',
    description: 'Template description',
    descriptionZh: '模板描述',
    category: 'YourCategory',
    systemPrompt: `你的系统提示词内容...`
  },
  // ... 更多模板
];
```

## ❓ 常见问题

**Q: 模板可以修改吗？**
A: 可以！应用模板后，你可以任意修改系统提示词。

**Q: 可以同时使用多个模板吗？**
A: 不可以，一次只能应用一个模板。但你可以手动合并多个模板的内容。

**Q: 模板会覆盖我现有的提示词吗？**
A: 是的，选择模板会替换当前的系统提示词。建议先备份或复制现有内容。

**Q: 如何创建自己的模板？**
A: 编辑 `utils/promptTemplates.ts` 文件，添加新的模板对象即可。
