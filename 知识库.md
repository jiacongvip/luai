# 智能体知识库

## 四类视频定义与模板
- 流量型：目标=注意力最大化；指标=前3秒停留、完播、互动、分享；特征=强娱乐/情绪/争议、泛垂直破圈、社交属性强；结构=开头认知冲突/情绪冲击→简单立场/解决→互动收束“你站哪边”；案例=争议/泛垂直/情绪钩子。
- 人设型：目标=信任与情感连接；指标=情绪反馈、关注增长、主页停留、主动搜索；特征=聚焦“四感”(价值/收获/共鸣/认同)、真实感、连续性、展示脆弱；结构=真实场景→个人经历+思考→启发式收束；案例=创业反思、专业人自处、价值观选择。
- 转化型：目标=驱动行为；指标=分层转化完成率（点赞/收藏/评论/主页/私信/购买）；特征=行为设计而非硬广，行动理由+路径清晰，降低阻力，损失厌恶加成；结构=精准筛选→不得不行动的理由→简化步骤，多入口转化；案例=技巧收藏、评论话题、主页承接。
- 混合型：目标=流量+转化平衡；指标=停留/互动 + 转化漏斗；特征=流量“糖衣”+转化“药丸”，转化顺其自然；结构=0-15s流量钩子→15-45s价值过渡→45-60s软转化→60s+深度转化；案例=挑战/剧情引入+价值过渡+橱窗/群私信等埋点。

## 类型决策逻辑（配比参考）
- 粉丝<5k：流量40/人设30/转化10/混合20，理由=快速测试模型+积累粉丝。
- 5k-50k：流量25/人设35/转化20/混合20，理由=深化人设+设计行为路径。
- ≥50k：流量20/人设25/转化30/混合25，理由=平衡热度与商业价值。
- 即时需求修正：急需涨粉→流量+20，混合+10，转化-15，人设-15；新品推广→转化+25，混合+15，流量-20，人设-20。配比最后归一到100%。

## 创作指令（简表）
- 流量型：前3秒认知意外/情绪冲击；留争议但不碰红线；泛人群话题破圈；互动话术激发站队。
- 人设型：聚焦1-2个“四感”；强调真实与脆弱；评论区延伸人设；收束用“希望对你有启发”。
- 转化型：设计看到→兴趣→行动的路径；多入口（赞/藏/评/主页/私信）；让点赞收藏有理由；行动话术口语化。
- 混合型：钩子中埋需求种子；福利前置/故事化；对比反差促进行动；多层转化设计。

## 转化行为层级
- 一级：私信/购买/预约（深度）
- 二级：主页访问/看其他视频/收藏（中度）
- 三级：点赞/评论/分享（轻度）
- 四级：认知好感（认知层）

## 文案选题基础
- 价值锚点：用户看完得到的具体收益（省钱、省时、技能、共鸣）。
- 情绪钩子：好奇/焦虑/惊叹/向往/争议。
- 传播基因：利他、站队、展示自我、寻求共识。
- 成本框架：时间/金钱/制作成本可控。

## 选题检测与自动选题规则
- 选题已给出的判断：用户文本中出现“选题/标题/这条视频是/帮我写一条关于X的视频/我想拍X”且X为明确主题，即视为已有选题，直接依据该主题做脚本。  
- 未给选题时（只描述身份/行业/门店/客群等），按以下步骤自动生成候选选题：  
  1) 抽取行业画像：  
     - 行业/品类（如：男装、教育、餐饮）；  
     - 场景（如：线下门店、电商、直播间）；  
     - 城市/圈层信息（如：常州、本地商务圈）；  
     - 客群标签（如：商务男士、高净值、30-45岁）；  
     - 价格带/成本（如：客单2000-50000，高客单、品质型）。  
  2) 映射核心策略与元素：  
     - 人群明确 → 优先：框定人群、身份认同、二元对立；  
     - 高客单/高价 → 优先：成本策略、高价值展示、慕强崇拜、头牌元素；  
     - 线下门店 → 可加：猎奇窥探（幕后/试衣间体验）、结果可视化（前后对比）；  
  3) 组合为2-3个策略 + 2-3个选题元素，调用多元素组合模型生成≥3个选题标题。  
  4) 所有自动生成的选题需标注：对应视频类型、策略组合、元素组合，便于用户理解与二次选择。

## 多元素组合规则
- 可用元素：成本、人群、猎奇、最差、反差、怀旧、荷尔蒙、头牌。
- 组合数量：2-4个，1个核心驱动元素 + 1-3个辅助；禁用组合=最差+头牌、荷尔蒙+怀旧。
- 逻辑连接：必须满足 因果/对比/递进 至少其一。
- 价值校准：前3秒吸引力提升≥50%，覆盖≥2个兴趣圈层，触发≥2种情绪。
- 兼容提示：成本↔人群/反差/最差高配；反差↔怀旧/最差/头牌高配；猎奇↔荷尔蒙/头牌/反差高配。
- 模型：双元素锚定（60/40）、三元素金字塔（基础层人群/成本→中层反差/猎奇→顶层情绪/价值）、四元素叙事弧（猎奇/反差→人群/成本→最差/怀旧→头牌/价值）。
- 质检：逻辑清晰、受众一致、15秒内能传达核心、情绪过渡自然；避免元素堆砌。

### 行业示例：常州高客单商务男装实体店
- 画像拆解：  
  - 行业：高端商务男装线下门店；  
  - 城市：常州（可映射为新一线/长三角商务氛围）；  
  - 客群：本地商务男士、重视体面与品味；  
  - 客单：2000-50000，高客单、高决策成本；  
  - 目标：涨粉+吸引本地精准客流+提升到店转化。  
- 推荐基础策略组合：  
  - 人设/信任向：框定人群 + 身份认同 + 高价值展示；  
  - 转化向：成本策略 + 损失厌恶 + 直击痛点（穿错场合的尴尬/损失）；  
  - 流量向：对比反差（穿搭前后/路人vs商务精英） + 猎奇窥探（试衣间/量体过程）。  
- 推荐选题元素组合示例：  
  - [人群元素（常州商务男士） + 成本元素（高客单但高性价比） + 高价值展示] → 适合混合型/转化型；  
  - [人群元素 + 反差元素（前后形象反差） + 慕强崇拜] → 适合流量型/人设型；  
  - [成本元素 + 损失厌恶 + 警告避坑] → 适合转化型“避坑类选购指南”。

## 质量检查清单
- 开头3秒是否建立认同/震惊/好奇。
- 情绪节奏：15秒内有高潮，45秒内给缓解与转化提示。
- 行动路径：转化步骤≤3步，口语化且门槛低。
- 争议/猎奇是否避开红线、符合平台规范。
- 数据/案例是否具体，避免空洞口号。
- 对照所选策略的强制要素（如反常识需科学依据，损失厌恶需具体损失描述）。
