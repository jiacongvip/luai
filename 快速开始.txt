==========================================
  Nexus AI - 快速开始指南（小白版）
==========================================

📋 第一步：准备环境变量

1. 在项目文件夹中找到 ".env.example" 文件
2. 复制它，改名为 ".env"
3. 打开 ".env" 文件，找到这一行：
   GEMINI_API_KEY=your-gemini-api-key-here
4. 把你的 Gemini API Key 替换上去
   （如果没有，去 https://aistudio.google.com/apikey 获取）

==========================================

🚀 第二步：一键启动

Mac/Linux:
  双击运行 "一键启动.sh" 文件
  或者在终端运行: ./一键启动.sh

Windows:
  使用 Git Bash 运行: bash 一键启动.sh

==========================================

⏳ 第三步：等待启动

脚本会自动：
  ✅ 启动数据库
  ✅ 启动后端服务
  ✅ 运行数据库迁移

等待 30-60 秒，看到 "✅ 所有服务已成功启动！" 就完成了

==========================================

🌐 第四步：访问应用

启动成功后，在浏览器打开：

  用户端: http://localhost:3000
  管理后台: http://localhost:3000/admin/login

==========================================

❓ 如果遇到问题

1. 确保 Docker Desktop 已经启动
2. 检查 .env 文件中的 API Key 是否正确
3. 查看日志: docker-compose logs -f

==========================================

🎉 完成！

现在可以开始使用了！
