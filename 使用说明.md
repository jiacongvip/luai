## ❓ 常见问题

### Q1: 提示 "Docker 未安装"
**解决**: 安装并启动 Docker Desktop

### Q2: 提示 "端口被占用"
**解决**: 
- 关闭占用 3000、3001、5432 端口的其他程序
- 或修改 `docker-compose.yml` 中的端口号

### Q3: 提示 "API Key 未设置"
**解决**: 检查 `.env` 文件中的 `GEMINI_API_KEY` 是否正确

### Q4: 数据库连接失败
**解决**: 等待 30-60 秒后重试，数据库需要时间启动

### Q5: 访问 http://localhost:3000 显示空白
**解决**: 
1. 确保前端也在运行：`npm run dev`（在另一个终端）
2. 或者检查浏览器控制台是否有错误
